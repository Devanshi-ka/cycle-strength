# Cycle & Strength

A cycle-aware fitness web application designed to help women make informed training decisions throughout different phases of their menstrual cycle.

The platform provides workout guidance, cycle phase visualization, and fitness recommendations through an intuitive and responsive user interface. The goal is to create a simple educational tool that helps users better understand how different phases of the menstrual cycle may influence training and recovery.

## Project Overview

Cycle & Strength was created to explore the intersection of fitness, health awareness, and frontend web development.

Many fitness applications focus on generic workout plans. This project aims to provide a more personalized experience by presenting workout suggestions and educational content based on different menstrual cycle phases.

The website emphasizes:

* Clean and modern user interface
* Responsive design for multiple devices
* Interactive user experience
* Educational fitness content
* Simple and accessible navigation

---

## Features

### Cycle Phase Calculator

Allows users to estimate their current menstrual cycle phase based on user input.

### Personalized Workout Guidance

Provides training recommendations tailored to different cycle phases.

### Interactive Cycle Visualization

Displays cycle phases in a visually engaging format to improve understanding.

### Educational Content

Explains the characteristics of each phase and how they may relate to training.

### Responsive Design

Optimized for desktop, tablet, and mobile devices.

### User-Friendly Interface

Simple navigation and clean layout focused on accessibility and usability.
